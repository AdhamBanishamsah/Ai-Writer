# Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Creating Icons

You can create icons using any image editor. The icons should represent the Ai Writer extension (e.g., a pen, text, or AI symbol).

### Quick Options:

1. **Use an online icon generator**: Search for "Chrome extension icon generator"
2. **Use a design tool**: Canva, Figma, or Adobe Illustrator
3. **Use an icon library**: Download from Flaticon, Icons8, or similar sites
4. **Create simple icons**: Use a text editor with emoji or simple shapes

### Temporary Placeholder

Until you have custom icons, you can:
1. Create simple colored squares as placeholders
2. Use any 16x16, 48x48, and 128x128 PNG images
3. The extension will work without icons, but Chrome will show a default icon

### Icon Requirements:

- Format: PNG
- Sizes: 16x16, 48x48, 128x128 pixels
- Background: Transparent or solid color
- Style: Should be recognizable at small sizes
